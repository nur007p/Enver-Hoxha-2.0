এই রিপোজিটরিটি একটি অটোমেটেড সিস্টেম যা কৃত্রিম বুদ্ধিমত্তা (AI) ব্যবহার করে প্রতি ২ ঘণ্টা অন্তর ফেসবুক পেজে ইউনিক ছবি এবং গল্পের মতো সুন্দর ক্যাপশন পোস্ট করে।

🚀 মূল ফিচারসমূহ
AI Image Generation: প্রতিটি পোস্টের জন্য নতুন ও ইউনিক ছবি তৈরি করে (FLUX.1 মডেল ব্যবহার করে)।

Storytelling Captions: প্রতিটি ছবির সাথে প্রাসঙ্গিক এবং রোমাঞ্চকর বাংলা ক্যাপশন তৈরি করে (Qwen/Llama মডেল ব্যবহার করে)।

Automated Workflow: গিটহাব অ্যাকশন (GitHub Actions) ব্যবহার করে সম্পূর্ণ অটোমেটেড।

Consistent Posting: প্রতি ২ ঘণ্টা অন্তর পোস্ট করার জন্য কনফিগার করা।

⚙️ সেটআপ নির্দেশিকা
আপনার গিটহাব রিপোজিটরিতে এই প্রজেক্টটি চালু করতে নিচের সিক্রেটগুলো (Secrets) যোগ করুন:

১. FB_PAGE_TOKEN: আপনার ফেসবুক পেজ এক্সেস টোকেন।
২. FB_PAGE_ID: আপনার ফেসবুক পেজ আইডি।
৩. HF_TOKEN: হাগিং ফেস (Hugging Face) এপিআই টোকেন।

কিভাবে যোগ করবেন:
গিটহাব রিপোজিটরির Settings > Secrets and variables > Actions-এ যান।

New repository secret বাটনে ক্লিক করে উপরের কি (Key) এবং আপনার টোকেনগুলো বসিয়ে দিন।

📅 অটোমেশন শিডিউল
এটি গিটহাব অ্যাকশনের মাধ্যমে নির্ধারিত সময়ে চলে। আপনি চাইলে .github/workflows/main.yml ফাইলটি পরিবর্তন করে সময় কম-বেশি করতে পারেন। বর্তমান সেটিংস:

YAML
schedule:
  - cron: '0 */2 * * *' # প্রতি ২ ঘণ্টা অন্তর
