# 🤖 FB Auto-Poster Bot (FLUX.1-schnell & Qwen2.5)

একটি সম্পূর্ণ অটোমেটেড পাইথন স্ক্রিপ্ট যা কৃত্রিম বুদ্ধিমত্তা (AI) ব্যবহার করে সম্পূর্ণ নিজে থেকে ইউনিক ঐতিহাসিক বা রহস্যময় টপিক খুঁজে বের করে, সেটির ওপর চমৎকার প্রম্পট ও বাংলা ক্যাপশন লেখে এবং হাই-কোয়ালিটি ছবি জেনারেট করে সরাসরি ফেসবুক পেজে অটো-পোস্ট করে। 

এটি ২৪/৭ সম্পূর্ণ ফ্রিতে ব্যাকএন্ডে রান করার জন্য **GitHub Actions** এবং **Cron-Job** ব্যবহার করে।

---

## 🌟 প্রধান বৈশিষ্ট্যসমূহ (Features)

- **🧠 সম্পূর্ণ অটোমেটিক টপিক জেনারেশন:** কোনো ফিক্সড লিস্ট ছাড়াই `Qwen/Qwen2.5-72B-Instruct` মডেল ব্যবহার করে প্রতিবার সম্পূর্ণ নতুন ও অনন্য ঐতিহাসিক/রহস্যময় আইডিয়া তৈরি করে।
- **🎨 হাই-কোয়ালিটি ইমেজ জেনারেশন:** বর্তমানের অন্যতম সেরা ওপেন-সোর্স ইমেজ মডেল `black-forest-labs/FLUX.1-schnell` (Hugging Face Hub-এর মাধ্যমে) ব্যবহার করে ক্রিস্পি এবং রিয়ালিস্টিক ছবি তৈরি করে।
- **📝 ইন্টেলিজেন্ট বাংলা ক্যাপশন:** ছবির প্রম্পটের ওপর ভিত্তি করে প্রাসঙ্গিক ইমোজি সহ আকর্ষণীয় বাংলা ক্যাপশন তৈরি করে।
- **⚡ অপ্টিমাইজড ও সুরক্ষিত:** সম্পূর্ণ প্রসেসটি Hugging Face ইকোসিস্টেমে স্থানান্তরিত করায় এটি অত্যন্ত দ্রুত কাজ করে এবং কোনো থার্ড-পার্টি এপিআই রেট-লিমিটের (`429 Too Many Requests`) ঝামেলা নেই।
- **🔒 সিকিউরড সিক্রেটস:** ফেসবুক পেজ টোকেন এবং হাগিং ফেস এপিআই কি সম্পূর্ণ সুরক্ষিত রাখতে GitHub Secrets ব্যবহার করা হয়েছে।

## 🛠️ টেকনোলজি স্ট্যাক (Tech Stack)

- **Language:** Python 3.11+
- **AI Libraries:** `huggingface_hub` (InferenceClient)
- **Image Processing:** `Pillow` (PIL)
- **Automation:** GitHub Actions & Cron-Job.org
- **API Integration:** Facebook Graph API

---

## ⚙️ সেটআপ গাইড (Setup Environment)

প্রজেক্টটি রান করার জন্য গিটহাবের **Settings -> Secrets and variables -> Actions**-এ গিয়ে নিচের **Repository Secrets** গুলো যুক্ত করতে হবে:

| Secret Name | Description |
|---|---|
| `FB_PAGE_TOKEN` | আপনার ফেসবুক পেজের Permanent Access Token |
| `FB_PAGE_ID` | যে ফেসবুক পেজে পোস্ট হবে সেটির ID |
| `HF_TOKEN` | Hugging Face-এর একটি ফ্রি Read Access Token |

## 🚀 যেভাবে কাজ করে (Workflow)

১. **টপিক চয়েস:** বট প্রথমে একটি ক্যাটাগরি (যেমন: প্রাচীন হারিয়ে যাওয়া সভ্যতা, রহস্যময় দুর্গ) সিলেক্ট করে AI দিয়ে একটি ইউনিক সাব-টপিক জেনারেট করে।
২. **প্রম্পট তৈরি:** ওই নির্দিষ্ট টপিকের ওপর ক্যামেরা অ্যাঙ্গেল ও লাইটিংয়ের ডিটেইলস সহ একটি ইমেজ প্রম্পট তৈরি হয়।
৩. **ক্যাপশন জেনারেশন:** প্রম্পটের সারসংক্ষেপ নিয়ে চমৎকার একটি বাংলা ক্যাপশন তৈরি করা হয়।
৪. **ছবি তৈরি:** FLUX মডেল দিয়ে ছবি জেনারেট করে সেটিকে মেমোরিতে প্রসেস করা হয়।
৫. **ফেসবুক আপলোড:** সবশেষে Facebook Graph API ব্যবহার করে ছবি ও ক্যাপশন পেজে সফলভাবে পোস্ট হয়ে যায়।
